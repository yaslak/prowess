# prowess

Personal dev toolkit for Laravel development with Claude Code — skills, commands, agents, hooks and MCP servers bundled in one plugin.

## Install

```bash
claude plugin install prowess
```

## Post-install

PHP language server (required for code intelligence):

```bash
npm install -g intelephense
```

## What's included

- **Skills** — Laravel best practices, TDD, debugging, frontend design, MCP server development, plugin development, and more
- **Commands** — Git workflows, PR review, code review, feature scaffolding, and more
- **Agents** — Specialized agents for code review, PR analysis, and development workflows
- **Hooks** — Security checks, session automation, and loop detection
- **MCP servers** — Context7, GitHub, Playwright, Serena, Laravel Boost
- **LSP** — PHP code intelligence via Intelephense
